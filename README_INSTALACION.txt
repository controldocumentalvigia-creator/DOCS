DASHBOARD SEMANAL DE CONTROL DOCUMENTAL

1. Guarda app.py y requirements.txt en la raíz del repositorio.
2. Instala dependencias:
   pip install -r requirements.txt
3. Ejecuta:
   streamlit run app.py
4. Cada semana carga el Excel actualizado desde el panel lateral.
5. Revisa las pestañas de vehículos y conductores.
6. Abre cada registro y usa el botón de copiar del bloque de mensaje.
7. Descarga el reporte semanal si necesitas guardar evidencia del seguimiento.

NOTA:
La hoja conductores del archivo revisado no tiene columna CELULAR. Si agregas una columna llamada CELULAR, TELEFONO, WHATSAPP o MOVIL, la aplicación la detectará automáticamente.
